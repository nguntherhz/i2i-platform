# Estrategia de Influencers Digitales

## Mercado

- Mercado de influencers virtuales: **$45 billion USD proyectado para 2030** (Grandview Research)
- Engagement rate de influencers virtuales: **3x mayor** que influencers humanos
- 37% de consumidores interesados, 37% desconfiados, 26% neutrales (Q3 2025)
- Las marcas pagan entre **$1,000-$10,000 por post** a influencers virtuales medianos

## Dónde está la oportunidad

### 1. Micro-influencers de nicho (10K-100K followers)
**Por qué:** Las marcas están migrando de mega-influencers a micro. Engagement más alto, más auténtico, más barato para las marcas. Es más fácil llegar a 50K que a 5M.

### 2. Nichos con mayor demanda y menos competencia

| Nicho | Oportunidad | Target Edad | Monetización |
|-------|------------|-------------|-------------|
| **Fitness lifestyle** | MUY ALTA | 18-28 | Marcas de ropa deportiva, suplementos, apps fitness |
| **Street fashion / Y2K** | ALTA | 16-24 | Fast fashion, vintage, thrift brands |
| **Wellness / Self-care** | ALTA | 20-30 | Skincare, yoga, mindfulness apps |
| **Student life / Campus** | MEDIA-ALTA | 17-22 | Mochilas, tech, cafeterías, lifestyle estudiantil |
| **Travel / Digital nomad** | ALTA | 22-30 | Airlines, hotels, travel gear, remote work tools |
| **Latina lifestyle** | MUY ALTA | 18-30 | Mercado sub-atendido, enorme audiencia |
| **K-beauty / Asian aesthetic** | ALTA | 16-28 | Skincare coreano, maquillaje, moda asiática |
| **Sustainable fashion** | MEDIA | 20-30 | Marcas eco, second-hand, slow fashion |

### 3. Plataformas prioritarias

| Plataforma | Prioridad | Por qué |
|-----------|----------|---------|
| **Instagram** | #1 | Visual-first, reels, stories. Core de influencers. |
| **TikTok** | #2 | Crecimiento explosivo, Gen Z, formato video corto. |
| **Pinterest** | #3 | Tráfico orgánico eterno, ideal para fashion/lifestyle. |
| **OnlyFans/Patreon** | #4 | Contenido premium exclusivo (no necesariamente NSFW). |

## Estrategia de Contenido por Persona

Para cada influencer se necesita un pipeline de contenido:

### Tipos de contenido a generar

| Tipo | Frecuencia | Workflow |
|------|-----------|---------|
| **Post feed** (outfit del día, escena lifestyle) | 3-4/semana | T2I + Super Catalog |
| **Variaciones de outfit** (mismo look, 3 opciones) | 2/semana | T2I → I2I change outfit |
| **Transformación** (gym look → brunch look) | 1/semana | I2I restyle |
| **Stories** (diferentes ángulos, behind the scenes) | Diario | I2I super catalog |
| **Reels/TikTok** (transición de outfits) | 2/semana | I2V (cuando esté listo) |
| **Colaboración con marca** (producto específico) | Según deal | I2I con prompt de producto |

### Consistencia de identidad

Para cada persona se necesita:
1. **Imagen base definitiva** (T2I con seed fijo)
2. **Banco de 20-30 variaciones** (Super Catalog con la imagen base)
3. **Guía de estilo** (colores, poses típicas, escenarios frecuentes)
4. **LoRA personalizada** (entrenar sobre las imágenes generadas para consistencia 95%+)
