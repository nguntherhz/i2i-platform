## Archivo
- `Qwen-Rapid-AIO.json` — Workflow en formato UI de ComfyUI

## Origen

## Diferencias con nuestros workflows

- Este no incluye LoRA F2P ni parámetros optimizados

## Uso
Arrastrar el archivo JSON al canvas de ComfyUI para cargarlo en la interfaz visual.
